# Fokir Portfolio Website

Fokir is a **modern, fully responsive personal portfolio website** built with **HTML5, CSS3, Bootstrap 5, FontAwesome, and JavaScript**. It showcases a personal profile, services, portfolio, client feedback, blog posts, and contact information in a clean and professional layout.

---

## Features

- **Responsive Navbar:** Sticky navbar with collapsible menu for mobile and tablet screens.  
- **Header Section:** Fullscreen hero section with typing animation for professional titles.  
- **About Me Section:** Personal information, profile image, CV download, and hire me buttons.  
- **Services Section:** Service cards with icons, hover effects, and smooth transitions.  
- **Portfolio Section:** Image gallery with overlay effects and project statistics.  
- **Client Feedback Section:** Carousel with star ratings for testimonials.  
- **Blog Section:** Posts with images and embedded videos.  
- **Contact Section:** Styled contact form (frontend only).  
- **Footer:** Social media icons and copyright information.

---

## Technologies Used

- **HTML5 & CSS3**  
- **Bootstrap 5** for responsive layout and components  
- **FontAwesome** for icons  
- **Google Fonts:** Roboto & Source Sans 3  
- **JavaScript** for Bootstrap interactions and animations
